### 2. 变量

变量的命名：

1. 首字符不允许为数字
2. 变量名中间不能有空格
3. 不能与 33 个 Python 保留字相同

命名技巧：

1. 变量名尽可能具有实际意义，表明数据的某种特性
2. 变量名有多个单词组成，用下花线 _ 连接多个单词
3. 驼峰体：多个单词组成，单词首字母大写（类名）
4. 常量所有字母均大写
